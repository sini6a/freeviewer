How it works

STUN only — free, works for ~80% of networks. Both peers discover their public IP and attempt a direct connection.

STUN + TURN — the TURN server relays traffic when a direct connection fails. Required for strict corporate firewalls, mobile carriers, and carrier-grade NAT.

This config is used by both the browser and all connected agents — you only need to set it once here.
